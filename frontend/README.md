# TriageAI — Frontend

React 18 + TypeScript + Vite admin dashboard for the TriageAI medical triage system.

## Development

```bash
npm install
npm run dev      # http://localhost:5173
```

## Build

```bash
npm run build    # output in dist/
```

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Recharts (analytics charts)
- Lucide React (icons)
- React Router v6

See the [root README](../README.md) for full project documentation.
